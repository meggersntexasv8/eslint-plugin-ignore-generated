# eslint-plugin-ignore-generated [![Build Status](https://travis-ci.org/zertosh/eslint-plugin-ignore-generated.svg?branch=master)](https://travis-ci.org/zertosh/eslint-plugin-ignore-generated)

Silently swallow [ESLint](https://eslint.org/) issues in files with `@generated`.

## Install

```sh
$ npm install eslint-plugin-ignore-generated --save-dev
```

## Usage

Add `ignore-generated` to `plugins` in your ESLint config. That's it.
